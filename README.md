# Habit Tracker App

A modern habit tracking application built with **.NET MAUI (C#)**.

This project was created during my university break to help me track my habits, stay consistent, and visualize my daily progress.

---

## Features

- Add new habits
- Edit existing habits
- Delete habits with swipe gesture
- Daily progress tracking
- Habit completion statistics
- Streak tracking
- Local data persistence
- Clean and modern UI

---

## Tech Stack

- **C#**
- **.NET MAUI**
- **XAML**
- **Local storage using Preferences**
